# fisheries-ecology
Scripts, apps, and related resources to support projects conducted by the [Fisheries Ecology & Enhancement Research Program at Mote Marine Laboratory](https://mote.org/research/program/fisheries-ecology-enhancement/).

## Data
In order to run the preprocessor for the restoration monitoring app, you must have the following files in the data folder:
- ORMR_data/RP_%d%b%y_*.txt
- FWC_RP_Tagdata.csv
- Robinson Preserve Seine Sampling Y1, Y2, Y3, Y4.csv

In order to run the preprocessor for the stock enhancement app, you must have the following files in the data folder:
- Hatchery.Release.Final.csv
- Hatchery.recaps.all.csv
- PC-antenna-locations.csv
- releases_by_year_creek.csv
- se_creek_locations.csv
